# 📃 API

### 🚀 Deployment

Check the [Live Deploy](https://street-wear-ecommerce-api.herokuapp.com/api) (hosted by [Heroku](https://www.heroku.com/)).

### 💻 Run Locally

Go to the project folder whit

```
cd challenge-17-pre-entrega
```

Install dependencies whit

```
npm install
```

Run locally whit

```
npm run dev
```
